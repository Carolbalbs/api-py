Welcome to Scipo for non-natives!

Scipo is a set of computational tools built to help non-native students write scientific papers in English. (integrated tools with the aim of helping students to write summaries and introductions of academic texts in the area of ​​Science, Technology and Innovation in English),it is based on the analysis of texts from the STEM area.!

Aqui,Você encontrará suporte para estruturar seus textos de acordo com as diretrizes da "boa redação" propostas pela literatura especializada e também poderá consultar nossa base de exemplos autênticos de resumos, introduções, metodologias, resultados,conclusões de discussões e conclusões de artigos de várias subáreas desta ciência.

  • Para começar a escrever um resumo (abstract), selecione a opção Abstracts no menu superior.
  • Para começar a escrever uma introdução, selecione a opção Intros.
  • Para começar a escrever uma metodologia (materiais e métodos)selecione a opção Metodologias.
  • Para começar a escrever um resultado, selecione a opção Resultados.
  • Para começar a escrever uma discussão, selecione a opção Discussões.
  • Para começar a escrever uma conclusão, selecione a opção Conclusões.
